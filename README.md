# 🎲 Dice Game

My first Python project! A simple dice game built with Python and tkinter.

## About
This is my first GUI application built while learning Python.
Two dice roll with animation and sound effects when you click.

## Features
- Animated dice rolling (20 frames)
- Sound effects on click
- Two dice simultaneously
- Fixed window size

## Technologies
- Python 3
- tkinter (GUI)
- winsound (sound)
- pyinstaller (build to .exe)

## How to run
1. Download all files
2. Run `tkinter_game.exe`

Or run from source:
python tkinter_game.py
